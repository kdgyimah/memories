# Express/Node Server with MongoDB Backend

## Start nodemon
```sh
npm start
```
This will make nodemon _watch for changes_ and it usually runs on `port 5000` or wherever the `process.env.PORT` parameter is configured to

## Supporting technologies
### MERN
> * Backend
> 1. [x] Express
> 2. [x] MongoDB
> 3. [x] Node
